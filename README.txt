David Pan | Portfolio
A professional portfolio website showcasing computer science projects, technical skills in firmware and full-stack development, and professional experience.

Built using the Massively template by HTML5 UP.

Sections
Intro: High-level summary of qualifications and contact information.

About Me: Detailed breakdown of technical skills, including Firmware Development, Python Automation, and Debugging/Testing.

Work Experience: Chronological history of professional roles including Vodafone and Angelo's Fish & Chips.

Notable Projects: A 2-column grid featuring full-stack applications, data visualization dashboards, and algorithmic console applications.

Certifications: Focused list of cybersecurity and technical credentials.

Technical Details
Layout: Fully responsive, article-oriented design.

Interactivity: Utilizes Scrollex for scroll-based transitions and a parallax background implementation.

Redirection: Custom logic implemented in generic.html to ensure navigation links drive traffic back to the core project grid on the main page.
